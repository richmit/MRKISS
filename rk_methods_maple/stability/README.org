
This directory contains data and associated Maple worksheets for the Butcher tableaux used by [[https://github.com/richmit/MRKISS][MRKISS]].

   - ~stab.mpl~: Procedures to process tableaux files
   - ~stab_all.mw~:  Process all tableaux
   - ~stab_test.mw~: Test procedures
   - ~RFunc/~: Stability function components, roots, & poles (generated by ~stab.mpl~)
     - ~*-RNpoly#.txt~: Numerator polynomial for each method in a tableau
     - ~*-RNroots#.txt~: Numerator polynomial roots for each method in a tableau
     - ~*-RDpoly#.txt~: Denominator polynomial
     - ~*-RDroots#.txt~: Denominator polynomial roots
   - ~RPlot/~: Stability plots (generated by ~stab.mpl~)
     - ~*-astab.png~: Combined stability plots
     - ~*-stab.png~: Stability plot for first one or two methods in each tableau
     - ~*-star#.png~: Order star plot for each method in a tableau
